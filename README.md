# Banking-Customer-Churn-Analysis
Developed an interactive Power BI dashboard to analyze customer churn in the banking sector. Leveraged SQL for data exploration and Power BI for visualization to uncover key business insights, monitor KPIs, and support strategic decision-making.

## 📌 Project Overview

Customer retention is a major challenge in the banking industry, as acquiring new customers is often more expensive than retaining existing ones. This project analyzes customer churn using SQL and Power BI to identify the key factors influencing customer attrition and provide actionable business recommendations.

The dashboard enables stakeholders to monitor customer demographics, churn trends, account activity, and financial metrics through interactive visualizations, helping decision-makers improve customer retention strategies.

Banks often experience customer attrition due to various factors such as low engagement, poor customer experience, changing financial needs, and competitive offerings. Understanding why customers leave is essential for developing effective retention strategies.

This project aims to identify customer segments with higher churn rates, analyze behavioral and financial patterns, and provide data-driven recommendations to reduce churn and improve customer loyalty.
